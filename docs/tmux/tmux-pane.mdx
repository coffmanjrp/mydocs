# Tmux pane
| Command                          | Description                                                           |
|----------------------------------|-----------------------------------------------------------------------|
| `Control + g /`                  | Divide pane horizontal                                                |
| `Control + g -`                  | Divide pane vertical                                                  |
| `Control + g x  or  Control + d` | Discard pane                                                          |
| `Control + g z`                  | Zoom in/out pane                                                      |
| `Control + g o`                  | Move cursor to next pane                                              |
| `Control + g ARROW`              | Move cursor to any pane                                               |
| `Control + g {`                  | Swap the pane to forward                                              |
| `Control + g }`                  | Swap the pane to backward                                             |
| `Control + g space`              | Change the layout of the panes                                        |
| `Control + g q`  => number       | Display the pane number then move the cursor the to the selected pane |
