# Tmux session
| Command                       | Description                |
|-------------------------------|----------------------------|
| `Control + g d`               | Detach from tmux           |
| `tmux ls`                     | List tmux sessions         |
| `tmux -a -t SESSION_NAME`     | Attach to tmux             |
| `Control + g s` => `0` or `1` | Select form tmux sessions  |
| `tmux kill-server`            | Shatdown tmux              |
| `Control + g t`               | Display time               |
