# Tmux window
| Command                          | Description                           |
|----------------------------------|---------------------------------------|
| `Control + g c`                  | New window                            |
| `Control + g n`                  | Move to next window                   |
| `Control + g p`                  | Move to previous window               |
| `Control + g NUMBER`             | Move to the window selected by number |
| `Control + g &` or `Control + d` | Discard window                        |
| `Control + g ,`                  | Rename window                         |
