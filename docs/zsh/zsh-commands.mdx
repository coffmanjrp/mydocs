# zsh commands
The following table lists some of the commands that can be used in zsh.
## Command lines

| Command      | Description                                                                                                                                                         |
|--------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `!<command>` | Allows you to execute the most recent command that starts with the specified command (ex. "`!vim`" would execute the most recent command that started with "`vim`") |
| `!!`         | Allows you to execute the previous command that you entered in the shell.                                                                                           |

## Shortcuts

| Command     | Description                                                                         |
|-------------|-------------------------------------------------------------------------------------|
| control + r | Allows you to search through your command history to find a specific command.       |
| control + a | Will move the cursor to the beginning of the current line.                          |
| control + k | Will delete all characters from the current cursor position to the end of the line. |
| control + e | Will move the cursor to the end of the current line.                                |
| control + b | Moves the cursor one character to the left.                                         |
| control + f | Moves the cursor one character to the right.                                        |
| control + d | Will delete the character at the current cursor position.                           |
