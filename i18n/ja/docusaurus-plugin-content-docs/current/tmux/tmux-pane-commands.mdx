# Tmux ペインコマンド
Tmux では、1 つのターミナルウィンドウを複数のペインに分割し、1 つのウィンドウで複数のコマンドやプログラムを実行することができます。ここでは、よく使われる Tmux のペインコマンドとその解説を紹介します。
:::info 
このページに記載されているコマンドは、Tmuxの設定によりカスタマイズされています。詳しくは、[Plese Click Here](/mydocs/tmux/tmux-settings) を参照してください。
:::

| コマンド                               | 概要                                                                                                                                                                                                       |
| -------------------------------------- | -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `Control + g /` | このコマンドは現在のペインを水平方向に2つのペインに分割することができます。                                                                                                                             |
| `Control + g -` | このコマンドは、現在のペインを縦方向に2つのペインに分割することができます。                                                                                                                               |
| `Control + g x` または `Control + d` | このコマンドは、現在のペインを破棄することができます。                                                                                                                                                           |
| `Control + g z` | このコマンドは、ペインを拡大・縮小することができます。あるペインをズームインすると、そのペインがウィンドウ全体を占め、他のペインは非表示になります。ズームアウトすると、ペインは元の状態に戻ります。|
| `Control + g o` | カーソルを次のペインに移動させるコマンドです。                                                                                                                                                |
| `Control + g ARROW` | カーソルを任意の方向のペインに移動するコマンドです。                                                                                                                                    |
| `Control + g {` | 現在のペインと左隣のペインを入れ替えるコマンドです。                                                                                                                            |
| `Control + g }` | このコマンドは、現在のペインと次のペインを右側にスワップします。                                                                                                                           |
| `Control + g space` | このコマンドは、ペインのレイアウトを変更することができます。                                                                                                                                                  |
| `Control + g q` => number | このコマンドはペイン番号を表示し、選択されたペインにカーソルを移動させることができます。|                                                                                                            |
