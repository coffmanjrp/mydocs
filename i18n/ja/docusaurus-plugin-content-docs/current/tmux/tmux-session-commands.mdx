# Tmux セッションコマンド
Tmux は端末のマルチプレクサで、一つの端末ウィンドウを複数のペインに分割したり、複数のセッションを切り替えたりすることができます。ここでは、よく使われるTmuxのコマンドとその解説を紹介します。
:::info 
このページに記載されているコマンドは、Tmuxの設定によりカスタマイズされています。詳しくは、[Plese Click Here](/mydocs/tmux/tmux-settings) を参照してください。
:::

| コマンド                                             | 概要                                                                                                                                   |
| ---------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Control + g d` | このコマンドは現在のTmuxセッションから切り離すことができます。つまり、ターミナルで他の作業をする間、セッションはバックグラウンドで実行され続けることになります。後でセッションに再接続するには、tmux attach コマンドを使用します。                                                                                                                                                       |
| `tmux ls` | このコマンドは現在実行中の全てのTmuxセッションをリストアップします。セッション名とその状態（アタッチされているかデタッチされているか）が表示されます。これは、どのセッションが現在アクティブであるかを確認し、後で特定のセッションにアタッチするのに便利です。                                                                                                                                                               |
| `tmux -a -t <SESSION_NAME>` | このコマンドは特定のTmuxセッションに名前を付けてアタッチすることができます。`a` フラグは Tmux が既に他の端末に接続している場合でも接続するように指示し、`-t <SESSION_NAME>` フラグは接続するセッションの名前を指定します。                                                                                                                                                 |
| `Control + g s` => `0` or `1` | このコマンドは異なるTmuxのセッションを切り替えることができます。Control + g s コマンドに続く数字 (`0` または `1`) は、切り替えるセッションを指定します。例えば、`Control + g s 0` はセッション `0` に、`Control + g s 1` はセッション `1` に切り替わります。このコマンドは複数の Tmux セッションを起動しており、それらのセッションを素早く切り替えたい場合に便利です。|
| `tmux kill-server` | このコマンドは Tmux サーバーをシャットダウンすることができます。実行中のセッションをすべて終了し、Tmux を終了します。このコマンドは、実行中のすべてのTmuxセッションを終了し、Tmuxを完全に終了させたい場合に便利です。                                                                                                                                                                                                      |
| `Control + g t` | このコマンドはTmuxのステータスバーに現在の時刻を表示させることができます。Tmuxで作業中に時間を把握したいときに便利なコマンドです。                                                                                                                                                                                                                                                |
