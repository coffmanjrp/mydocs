# Tmux 設定のカスタマイズ
設定ファイルをカスタマイズすることで、Tmuxの使用感をコントロールすることができます。
### プレフィックスキーの変更
デフォルトでは、Tmuxのプレフィックスキーは "b "です。しかし、以下の手順で別のキーに変更することができます。
1. 1. ホームディレクトリに "`.tmux.conf`" というファイルを作成します。
2. 2. テキストエディタでこのファイルを開きます。
3. 3.このファイルで以下の設定を行います。
```
C-dのバインドを解除する
set -g prefix C-g
bind C-g send-prefix
```
これで、プレフィックスキーがデフォルトの "b "から "g "に変更されます。
### 分割ペインキーのカスタマイズ
divide pane のキーバインドを設定するには、`.tmux.conf` ファイルに以下の設定を追加します。
```
bind ￭ split-window -h
bind - split-window -v
```
これにより、ペインを水平に分割する場合は"Ⓐ"、垂直に分割する場合は"-"をそれぞれ使用できるようになります。
### ペイン番号の表示時間を変更する 
ペイン番号の表示時間を調整するには、ファイルに以下の行を追加します。
```
set -g display-panes-time 3000
```
これで、ペイン番号が3秒間表示されるようになります。
### Vim モードからの脱出をより速く
Vimモードの脱出時間を早くする設定は、以下の行をファイルに追加してください。
```
set -s エスケープタイム 0
```
### Shift + 矢印キーでウィンドウを切り替える
Shift + 矢印キーでウィンドウを切り替えるには、ファイルに以下の行を追加してください。
```
bind -n S-Left 前のウィンドウ
bind -n S-Left next-window (右のウィンドウ)
```
変更を保存し、Tmux を再起動すると、新しい設定が有効になります。
