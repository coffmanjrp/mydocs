# Tmux ウィンドウコマンド
Tmux では、セッション内に複数のウィンドウを作成し、切り替えることができます。各ウィンドウには、1つまたは複数のペインが含まれます。以下は、よく使われる Tmux のウィンドウコマンドとその説明です。
:::info 
このページに記載されているコマンドは、Tmuxの設定によりカスタマイズされています。詳細は、[Plese Click Here](/mydocs/tmux/tmux-settings) を参照してください。
:::

| コマンド                             | 概要                                                                  |
| ------------------------------------ | ----------------------------------------------------------------- |
| `Control + g c` | このコマンドは新しいウィンドウを作成することができます。|
| `Control + g n` | このコマンドは次のウィンドウに移動します。              |
| `Control + g p` | このコマンドは前のウィンドウに移動します。          |
| `Control + g <NUMBER>` | このコマンドは番号で選択されたウィンドウに移動します。|
| `Control + g &` または `Control + d` | このコマンドは現在のウィンドウを破棄します。           |
| `Control + g ,` | このコマンドは現在のウィンドウの名前を変更します。            |
